**Contact Management REST API**

**PROJECTS AIM:**
Manages users and their contact information (phone, email, address).
Demonstrate concepts like API design, authentication, database management, and structured error handling.

**TECHNOLOGIES:**

**Languages:** python

**Frameworks:** Flask

**Database:** SQLite

**Tools:** Git, GitHub, Vs code, postman


**LAUNCH:**
Implemented full CRUD functionality with relational database design using SQLite.

Secured endpoints with API key authentication and input validation.

Designed standardized JSON responses and error handling for reliable API behavior.

Managed development using Git with iterative commits for version tracking.
